# DualControlVault

**DualControlVault** is a multi-owner smart account implementation designed for the Base network, focusing on shared asset management and secure batch transaction execution.

## 🔗 Contract Details
* **Network:** Base Vibenet (Testnet)
* **Contract Address:** [0xaF22986aD254cEC96CFd00e94A769183f479f655]

## 🚀 Key Features
* **Dual-Owner Logic:** Shared administrative control between two authorized addresses.
* **Batch Execution:** Optimized multi-recipient transaction handling for gas efficiency.
* **Asset Management:** Secure deposit and withdrawal functionality.

## 🛠 Tech Stack
* Solidity ^0.8.20
* Remix IDE
* Rabby Wallet (Base Vibenet)

## 📋 Status
Deployed on Base Vibenet for infrastructure testing and multi-sig wallet prototyping.
